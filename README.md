# YOUMAN APP 🚀

A premium health & discipline tracking application designed to optimize male performance and habits.

## Features ✨

- **Daily Protocol**: Structured 30-day discipline program.
- **Knowledge Base**: Curated content to improve health and testosterone levels.
- **Progress Tracking**: Streak system and level-based progression.
- **Store**: Integration for premium health kits and equipment.
- **Premium UI**: Sleek, glassmorphism design with fluid animations.

## Tech Stack 🛠️

- **Frontend**: React 19 + Vite
- **Styling**: Vanilla CSS with Glassmorphism
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Database/Auth**: Supabase
- **Hosting**: Vercel

## Getting Started 🚀

1. **Clone the repository**:
   ```bash
   git clone https://github.com/hagiaya/youmanapp.git
   cd youmanapp
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

## Deployment 🌐

### GitHub
Push changes to the `main` branch to trigger auto-deployment if connected to Vercel.

### Vercel
1. Connect your GitHub repository to Vercel.
2. Add the environment variables (`VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`) in the Vercel dashboard.
3. Deploy!
